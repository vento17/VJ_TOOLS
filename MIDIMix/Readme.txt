MIDIMIX

This object was realized to fast link
knobs, sliders and buttons of the 
Akai MIDIMix Usb Controller.

All Values are scaled from 0-127 to 0-1

Drag and drop them!

How to setup the controller:
- Top menu
	Dialogs/Midi Device Mapper
- Create New Mapping
	In Device : MIDI Mix
	Out Device : MIDI Mix
- Done!

On Custom Parameters
turn off the cover texture to reduce the Cook Time.
(half ms instead of 1 ms, on my laptop)
